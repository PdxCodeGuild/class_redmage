About

Threshold is a workout webapp created by Cory Morrissey for his final project at PDX Code Guild. It uses data pulled exclusively from Bodybuilding.com.

How To Use This App

Home

Contains three default workouts spanning over three days. Used to get a feel of what the website offers.

Menu > All Exercises

Contains all exercises available, select the muscle then select the exercise for full details.

Menu > User's Workouts

Here is where you can create your workouts. Once the workout has been created, you are able to set exercises to your respective workouts.

What Muscle Groups To Workout Together

The chest, shoulders, and triceps. The back and the biceps. Hamstrings (biceps), calves and the glutes.

Day 1: Training the chest, shoulders, and triceps muscles.

Day 2: Back and biceps exercises.

Day 3: Leg exercises.

This is a rough guide, but using this information the user should be able to create workouts in a matter of minutes. For more details on building workouts, I like the link below.
