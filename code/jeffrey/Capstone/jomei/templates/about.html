Welcome to jomei! A experiment in spreading light in the world by adding context to locations.

Jomei is the capstone project following a 12-week full stack coding bootcamp covering python, html and css, javascript, and utilizing the django python web framwork.

Version 1 of this project was to create a 2D mapping and point creation web application to demonstrate plotting markers and the relationship between the markers with the user's origin.

Future iterations aim to leverage native application frameworks to more easily interact with smart phone/device cameras so that once markers are plotted, then they can be visualized in an augmented reality experience in relation to the user.
